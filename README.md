# Chess ML Application

A full-stack chess application that learns from historical chess games and allows players to train the model through gameplay.

## Tech Stack
- **Backend**: PostgreSQL, Python
- **ML/AI**: TensorFlow, python-chess
- **Frontend**: Flask (planned)
- **Data**: Lichess game database

## Project Structure
```
chess_project/
├── notebooks/          # Jupyter notebooks for analysis
├── app/               # Flask application
├── models/            # Trained ML models
├── data/              # Scripts for data processing
└── sql/               # Database schemas
```

## Setup Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Set up PostgreSQL database
3. Run data pipeline notebooks
